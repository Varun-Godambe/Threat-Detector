AI-Powered Analysis Capabilities

ThreatSight deploys a suite of specialized AI agents to perform deep analysis on files and URLs, providing comprehensive security insights for both everyday users and cybersecurity professionals.

URL Analysis

Designed for simplicity and clarity, this feature allows anyone to check if a link is safe before clicking. The URL Analysis Agent provides:

  • Clear Verdict: A simple, color-coded result (Safe, Suspicious, or Malicious) that is easy to understand at a glance.
  • Plain-Language Summary: A brief explanation of why the URL received its verdict, avoiding technical jargon.
  • Technical Reasons: For those interested, a list of specific reasons for the verdict, such as blacklist status, domain reputation, or certificate validity.

File Analysis: For All Users

When any file is uploaded, the Static Analysis Agent performs a universal scan to identify potential threats, providing a clear summary suitable for all users.

  • Threat Level Assessment: The file is assigned a threat level from "Safe" to "Critical" based on the findings.
  • AI Agent Summary: A clear, plain-language explanation of what was found.
  • Technical Detections: A list of specific findings, such as disguised file signatures (e.g., a .txt file with an executable header), high entropy suggesting packed/encrypted code, and dangerous embedded commands. Simulated CVEs are included for context.

File Analysis: For Professionals (Log Files)

When a log file (.log, .txt) is uploaded, the Logfile Forensics Agent takes over to provide a professional-grade incident report.

  • Generated Vulnerability Report: A pre-formatted, copy-and-paste ready report section detailing the incident, key findings mapped to the MITRE ATT&CK framework, and actionable recommendations.
  • Threat Intelligence Correlation: A table of all IP addresses found in the log, checked against a simulated threat database to flag known malicious actors.
  • Reconstructed Attack Chain: A step-by-step visualization of the attack, from initial access to data exfiltration, making the incident easy to follow and document.
