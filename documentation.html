Real-Time Analysis Engine

ThreatSight employs a powerful, hybrid analysis engine to provide state-of-the-art security insights. It intelligently routes submissions to the best-suited analysis backend, ensuring comprehensive and accurate results for every type of query.

URL Analysis (Powered by VirusTotal)

Designed for simplicity and clarity, this feature allows anyone to check if a link is safe before clicking. When a URL is submitted, it is sent to the VirusTotal API for scanning.

  • Multi-Vendor Scanning: The URL is analyzed by over 70 different antivirus scanners and URL blacklisting services.
  • Clear Verdict: A simple, color-coded result (Safe, Suspicious, or Malicious) is provided for an immediate, easy-to-understand assessment.
  • Detailed Report: Users can view the specific results from each security vendor for a complete picture of the URL's reputation.

File Analysis: Malware & Executables (Powered by VirusTotal)

For any file that is not a plain text log or configuration file (e.g., .exe, .dll, .pdf, .docx, .zip), ThreatSight leverages the VirusTotal API for deep malware analysis.

  • Comprehensive Malware Scanning: The file is uploaded and scanned by the full suite of VirusTotal's security engines to detect known malware, trojans, and other threats.
  • Actionable Verdict: The results clearly indicate if the file is malicious, allowing users to take immediate action, such as deleting the file.

File Analysis: Logs & Configurations (Powered by Gemini AI)

When a text-based file is uploaded (e.g., .log, .txt, .json, .xml, .conf, .py, .sh), ThreatSight deploys a specialized **AI Forensics Agent**. This agent operates under the combined best practices of NIST, ACSC, and OWASP.

  • Context-Aware Analysis: The AI is prompted to act as a senior cybersecurity analyst, looking for patterns like repeated failed logins, unauthorized access attempts, hardcoded passwords, insecure settings, behavioral anomalies, and application-layer attack patterns.
  • Visual Dashboard: The AI returns its findings in a structured format, which are then displayed in a professional dashboard with summary cards and charts for an immediate overview.
  • Actionable Intelligence: Each finding includes a severity level, the relevant timestamp, and a clear, actionable recommendation.
